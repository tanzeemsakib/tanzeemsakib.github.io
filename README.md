# tanzeemsakib.github.io

Personal portfolio site. Built with vanilla HTML, CSS, and JavaScript.

## Stack

- HTML / CSS / JS — no frameworks
- Lenis for smooth scrolling
- DM Serif Display + DM Sans (Google Fonts)
- Hosted on GitHub Pages

## Structure

Assets (images, GIFs, resume) live in `/assets`. 
Everything else is a single `index.html`.
